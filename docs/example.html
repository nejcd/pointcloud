Pointcloud.py¶

This is demo of small python pacakge for working with point clouds. IT was developed for faster and easyier researching and prototypeing new neural networks for point cloud classifcation.

Features:

  • support of multi tile pointclouds
  • subsampling
  • statistics
  • retiling
  • saving
  • generating train / test split

Todos:

  • generating balanced train samples
  • different readers (now it is possiable to read only .las files with use of laspy as backend)

We dan generate new project from above settings. In project class are all meta data needed. Project can has many pointclouds and each point cloud can be tiled. If we have large project of whoel countires data can not be stored in single file. There for it is tiled. When we are adding new pointcloud it can be from single or multi tiles. For each tile we have to caluclate bounding box.

We have two options for bounding box calculation:

  • Calulcation from file name (lot of times tile grid is reqular sized grid. Coordinates of one corner are usualy spcified in filename. There for we can reconstruct bounding box from X, Y and grid size) - fast
  • Caluclation of convex hull around points - slow

We can save project and later loaded it so this calulcation does not need to be done every time.

    print('Processing Polygons')

    for i, data in enumerate(names_polygons):

        sys.stdout.flush()
    
        sys.stdout.write('\r Adding tiles {0}/{1}'.format(i + 1, len(names_polygons)))
        sys.stdout.flush()
        pointcloud.add_tile(data['name'], data['polygon'])
